# Todo-App

This is a simple todo app.

## Development Approach
- Test Driven development

## Language
- Typescript

## Tools
- Build tool ( vite )
- Testing ( vitest, testing-library)
- Framework ( Vue.js v3 )
- State Management ( Pinia )
- CSS (SASS)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run tests
```
npm run test
```
